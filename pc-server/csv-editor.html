Joan of the City — Audio Cue CSV Editor

Edit audio_cues.csv — isolate cues by A/B/C track, edit inline, export with auto version bump.

How to use this

1 — Open your file

Click Import CSV… in the File panel and pick audio_cues.csv. If you launched with start-csv-editor.bat, it can auto-load via Fetch too.

2 — Find what you want to edit

On the Edit tab, use Select cue to focus on one cue (e.g. 211). Optionally:

  • Cue range shows several cues at once.
  • Prefix (A/B/C) and Type (SQ/VQ) narrow which rows show.
  • Text search looks across every field for words like “communion”.

Or scroll the Full CSV at the bottom. SQ rows are tinted teal, VQ rows orange.

3 — Make changes

  • Click a cell to edit. Action is a dropdown (with a Custom… option). Volume is a slider.
  • Drag the grip ⠿ on any row to reorder it.
  • Press + Add next to a track (A/B/C) to add a new row for that cue.

4 — See how it plays out

The Visualize stems tab shows when each audio file starts, holds, or stops across cues — pick stems on the left and the timeline appears on the right. The ▶ button plays a stem.

5 — Save

Click Export → audio_cues.csv. You'll be asked whether to bump version numbers, and the previous file is kept as audio_cues_v<n>.csv as a safety net.

If something goes wrong

Use ↶ Undo (or Ctrl+Z outside a text field) to take back recent changes. Your work is also auto-saved in the browser, so closing the tab won't lose it.
